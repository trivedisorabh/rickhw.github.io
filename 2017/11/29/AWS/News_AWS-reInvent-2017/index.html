AWS reInvent 2017 News

整理 AWS re:Invent 2017 發佈的新東西！

New Services

Preview

  • Amazon Elastic Container Service for Kubernetes
    • Amazon EKS integrates IAM authentication with Kubernetes RBAC (the native role based access control system for Kubernetes) through a collaboration with Heptio.
    • Amazon EKS is available in Preview.
  • In The Works – Amazon Aurora Serverless
  • Amazon Neptune – A Fully Managed Graph Database Service

GA

  • Amazon Comprehend – Continuously Trained Natural Language Processing
  • Introducing Amazon Translate – Real-time Language Translation
  • Amazon Kinesis Video Streams – Serverless Video Ingestion and Storage for Vision-Enabled Apps
  • Welcoming Amazon Rekognition Video: Deep-Learning Based Video Recognition
  • Amazon SageMaker – Accelerating Machine Learning
  • Introducing AWS Fargate – Run Containers without Managing Infrastructure
    • It’s a technology that allows you to use containers as a fundamental compute primitive without having to manage the underlying instances.
    • Fargate is available today in the US East (Northern Virginia) region.
  • Amazon DynamoDB Update – Global Tables and On-Demand Backup
  • Amazon MQ – Managed Message Broker Service for ActiveMQ: Apache ActiveM
  • Introducing AWS AppSync – Build data-driven apps with real-time and off-line capabilities: GraphQL
  • AWS Media Services – Process, Store, and Monetize Cloud-Based Video: OTT Killer?
  • Presenting Amazon Sumerian: An easy way to create VR, AR, and 3D experiences

New Features

Compute

  • T2 Unlimited – Going Beyond the Burst with High Performance
    • 這是一種給錢就做事的概念。
  • H1 Instances – Fast, Dense Storage for Big Data Applications
    • 最大 h2.16xlarge 64core, 256GiB, 16TB Local Storage.
  • M5 – The Next Generation of General-Purpose EC2 Instances
    • 最大 m5.24xlarge, 96core, 384GiB, 25Gbps, EBS-Optimized: 10000Mbps
  • Amazon EC2 Update – Streamlined Access to Spot Capacity, Smooth Price Changes, Instance Hibernation
  • Amazon EC2 Bare Metal Instances with Direct Access to Hardware

Storage

  • S3 Select and Glacier Select – Retrieving Subsets of Objects
    • Built-in Athena?!

Security, Identity, Compiance

  • Amazon GuardDuty – Continuous Security Monitoring & Threat Detection

Network

  • Keeping Time With Amazon Time Sync Service
  • AWS PrivateLink Update – VPC Endpoints for Your Own Applications & Services

Management

  • AWS Systems Manager – A Unified Interface for Managing Your Cloud and Hybrid Resources
    • 更好管理資產了
    • AWS 的 Ansible

IoT

  • Announcing Amazon FreeRTOS – Enabling Billions of Devices to Securely Benefit from the Cloud
    • 居然搞氣 RTOS, 居然放在 IoT ！！
  • Presenting AWS IoT Analytics: Delivering IoT Analytics at Scale and Faster than Ever Before
    • IoT 的整合分析，整合了自己的 Amazon Quicksight 和 Jupyter Notebooks.
