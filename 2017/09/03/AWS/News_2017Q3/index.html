AWS 2017Q3 New Features

整理 2017Q3 AWS 的新功能。

AI / ML

  • New – Amazon EC2 Elastic GPUs for Windows

System Operations / Infra

  • Amazon Virtual Private Cloud (VPC) now allows customers to expand their existing VPCs
  • New – Application Load Balancing via IP Address to AWS & On-Premises Resources: ALB 可以用 IP 指定 LB 導向的資源，意思就是不僅止於 VPC 裡面的，只要 ALB 可以連線的都可以掛在 ALB 後面，像是公司裡的機器、或者是其他透過 VPC Peering 的 VPC
  • New – Descriptions for Security Group Rules: 管理 Infra 很需要的的功能，我至少 submit 過四五次的 Feedback。。。終於。。。不過沒有 Network ACLs 很可惜。。
  • Monitoring Your NAT Gateway with Amazon CloudWatch: 實用！不用自幹了！
  • Amazon Aurora Fast Database Cloning
  • AWS Macie: 用 ML 做資安的新服務
  • VMware Cloud on AWS – Now Available
  • AWS Cost Explorer Update – Better Filtering & Grouping, Report Management, RI Reports
  • New – SES Dedicated IP Pools
  • New – VPC Endpoints for DynamoDB
  • New – Encryption of Data at Rest for Amazon Elastic File System (EFS)
