OS X 鍵盤問題

幾個 OS X 使用鍵盤的問題：

(在不改動系統設定的前提之下)

  1. 不習慣的就是右邊的 control 不見了。
    • 通常組合鍵，我是用平衡按法，也就是字母在左半邊，就用右邊的 ctrl；反之，也是。不過這個平衡按法，無法在 Mac 鍵盤上按，因為右邊的 Ctrl 不見了。
    • 多重複合 (三個按鍵以上): 像是 Word Selection: (Option + Shift + 方向鍵)，MBPR 的配置來講，右手按左右方向鍵，左手按 Option + Shift .. 很難按 … 因為 Option 和 Shift 的距離太近，手會很難按，不符合人體工學。 Windows 104 Key 則是 Alt + Shift + 方向鍵， Alt + Shift 的距離沒有太近，算剛好。
  2. Mac 的方向鍵太小：像是下常常按不到，或者按到了，卻沒反應 (試過好幾台，都有同樣的問題)。按鍵太小沒有存在感。
  3. 預設的 Function Key (亮度、Mission Control) 實用性不大，用 Touch Pad 就可以了。
  4. 沒 pgdn/up/home/end 可用，這些都要搭配 fn 才行，有時候我需要 Ctrl + F9 (Debug)，就要按三個 …
    • 選擇一行: Cmd + Shift + Right or Left Windows 104: Shift + Home or End
    • 選擇一頁: 不知道 … Windows 104: Shift + Page
  5. Mac 鍵盤敲起來爽度沒 think pad 爽，有點太幼秀的感覺。。。。

關掉 Mac 預設的 Fu Key (亮度/Mission control/音量 …etc)

  • Perference -> Keyboard -> Check “Use all F1, F2, etc. keys as starndard function keys.”
