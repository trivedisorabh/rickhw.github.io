AWS Longer Resource ID

來自: Longer EC2 & EBS Resource IDs Coming in 2016

AWS Resource ID 會變長，從 8 chars, 變成 17 chars, 舊的 id 不變，新的 resource 長度會變長, SDK 不影響.

  • 影響範圍：EC2/EBS/Snapshot
  • 時間: 2016/01 - 12 ，提供新舊 ID 轉換的選項.
  • AWS: CLI/SDK/PowerShell 都已經支援 Longer resource IDs.
