Extend partition size for CentOS in VMWare

MongoDB needs large disk space to storage database and log. It happened a lots of time in my VMWare machines, and I tried to fix. The symptom is as below:

The error shows make at least 3.3GB, check my storage status:

Environment

  • CentOS 6.4 64bit
  • Storage: 20G -> 40G

Summary

  1. create a new partition
  2. specify partition type as 8e
  3. reboot
  4. extend the partition size

pvcreate /dev/sda3

vgextend vg_oplab /dev/sda3

lvextend -L+19.99G /dev/vg_oplab/lv_root

resize2fs /dev/vg_oplab/lv_root
