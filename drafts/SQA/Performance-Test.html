效能測試

測試的種類

整理一些軟體 註 的效能測試，

註：這邊的 軟體 指的是 Server Side 這種的應用程式的效能而言。

  • Stability (穩定性): 時間性, 資料完整性 (integrity)
  • Reliability (可靠性): 瞬間,
  • Capacity (容量) / Load Testing (負載測試): 進來的東西都是對的，大量
    • 找出系統的 Memory Leak
    • 找出系統屬於 CPU-Bound, Memory-Bound ..
    • 找出系統的 Benchmark，可以數學方法推算的 base-line
  • Stress Test (壓力測試): 輸入的有對有錯，大量
    • 著重驗證系統的可恢復性 (Recoverability)
