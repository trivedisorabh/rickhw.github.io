AWS Certified DevOps Engineer Pro - 考試心得

整理 AWS Certified DevOps Engineer Pro 考試心得。

考試資訊

  • 官方考試資訊：AWS Certified DevOps Engineer - Professional
  • 考試範圍：BluePrint
  • 考題：80 題
  • 作答時間：170 分 (2 小時 50 分)
  • 考試費用：300 USD，練習考試為 40 USD

整體來說，都比 Associate Level 多一倍，平均每題大概只有兩分鐘的時間作答，而且題目都很長，英文閱讀不好會很慘。

    • Elastic Beanstalk (eb, PaaS)
    • OpsWorks (Chef)
    • AWS CLI, SDK, API
  • Domain 2: Monitoring, Metrics, and Logging 20% == 16
    • CloudWatch
    • CloudWatch Log
  • Domain 3: Security, Governance, and Validation 10% == 8
    • Cloudtrail
    • IAM
  • Domain 4: High Availability and Elasticity 15% == 12
    • AutoScaling
    • VPC

Provision

Provisioning 是 cloud 普及後出現的階段性任務，簡單說：「開機器」就是 provisioning，但現在一套系統通常不會只有「開一台機器」這麼單純，AWS 上常看到 VPC, SG, Subnet, ELB, S3, ASG, CloudFront, Route53, Redis, SES, SNS, CloudWatch … 等一堆要「準備好」、「串聯」， 然後才是輪到開發出來的應用程式的「部署」。這類的事情都可以自動化。常見的工具 AWS 以 CloudFormation 最完整，不過我覺的適合全新的案子，既有的系統用 cli or sdk 會比較適合。

再談專業

以前唸書時，有同學老是喜歡翹課去打籃球，有次翹課的太嚴重，老師就說了一段故事：

同學如果你很喜歡打籃球沒關係，那麼你的目標就要抓 Michael Jordan 那樣，如果不是，那麼你至少要做到，哪一年哪一場球、誰進球幾分、犯規幾次、最佳助攻。這樣才是專業，那你就可以不用來上課了。

以前修微處理機 (Micro-processor) 的課，老師第一堂課開宗明義就講，這堂課修完只是入門，只能說對 8051 這塊晶片知道一點皮毛，一塊錢都賺不到。

幾年前年，有學生私下寫信問我想專職走音樂的路，如何針對用吉他謀生。一開始我想到曾經寫過的這篇『十年 … 不過是入門而已 …』，後來我整理了『拿音樂當飯吃？』。

如果要變成 Cloud Engineer 的專家，專業的程度至少應該是這樣的：

  • 熟悉所有 AWS 的服務，能夠運用這些服務，實際解決任務的需求。
  • 知道其他類似的雲端服務，像是 Azure、GCP、SoftLayer …
