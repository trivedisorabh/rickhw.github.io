Study Notes - CloudFront

Architecting for the Cloud

Terminology

  • Edge Location:
    • Can read and put object
    • Object are cached for life of TTL.
    • clear cached will be charged.
  • Origin: S3 Bucket, EC2 Instance, ELB, Route53
  • Distribution:
    • Web Distribution
    • RTMP: Media

注意事項

  • 修改 CloudFront 設定，每次更新約要 10-20 分鐘

全站 CDN: CloudFront + S3 + Route53 (DNS) + SSL

  • 在 Route53 購買 DNS, 這裡用 abc.com 為例
    *
  • 規劃網站 DNS, ex: blog.abc.com
  • 在 S3 建立 Bucket, 名稱同 DNS: blog.abc.com
