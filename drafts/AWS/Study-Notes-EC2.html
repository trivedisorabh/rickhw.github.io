Study Notes - AWS EC2

整理 AWS EC2 的疑惑與心得。

EC2 (Elastic Compute Cloud) 是 AWS 的 Virtual Machine (虛擬機器)，設計的前提就是給 IT 人員使用，同時 EC2 也是 AWS 最重要的核心服務。

所以使用者必須瞭解『組裝』一台電腦所需要的知識，包含 CPU / Memory / Storage / Network 等概念。

Trouble shooting for EC2 Connection and Security Group

如果是對外無法正確連線, 連不出去, 無法更新 patch:

  • 確認 Security Group 的設定是正確, EC2 / ENI 都要確認. 特別是 ENI
  • 確認 EC2 的 Subnet 走的 Route Table, 有 NAT Gateway

如果是外面連不進來:

  • 確認 Security Group 的設定是正確, EC2 / ENI 都要確認.
  • 不管 EC2 是否掛在 ELB 底下
    • 確認 ELB 設定的 Subnet 走的 Route Table, 有 Internet Gateway
    • EC2 不是掛 ELB, 那麼就要 Attach EIP 到 ENI.

如果這些都檢查過沒問題，那麼就要往 ACL 檢查了。
