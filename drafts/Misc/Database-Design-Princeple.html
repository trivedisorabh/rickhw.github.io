資料的種類

根據存取的頻率分成靜態、動態

靜態的類別像是：

  • 使用者資料
  • 設定資料
  • 歷史資料
  • 存取記錄
  • 列舉 enums
  • 分析資料

動態資料像是：

通常是一堆 foregion key 的關聯表, 通常都是參考靜態表的.

  • 在製品
  • 設定類別
  • 關聯表

Prefix 命名規則

  • cfg_: data by user, user name, detail info, device info …
  • related_: the related by user
  • ps_: in process, 在製品
  • history:
  • log_: 系統 log, 建議寫到 disk
  • enum_: enum for system

對外 <–> 對內

人 (end user) –> 事 (process) –> 物 (system)

物 (system) –> 事 (event) –> 人 (admin/ops)
