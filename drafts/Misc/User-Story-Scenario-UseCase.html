User Story / Scenario, Use Case

User Story

特定角色的簡短使用流程. 重點在於 Who, What, Why 三個. 常見的例子結構就是:

A [type of user] wants to [do something] so that [outcome happens].

  1. As a (user type) I want to (action/feature) so that (reason)
  2. As a (user type) I want to (action/feature) because (reason)

User Story 的目的

User Scenario

類似於 User Story, 著重於流程 (Procedure) 的描述, 如果 User Story 是這樣：

As a (user type) I want to (action/feature) so that (reason)
