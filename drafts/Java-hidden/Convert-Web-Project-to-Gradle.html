Convert Web Project to Gradle

把我自己目前寫的一個 Web project 從 ANT build 改成 Gradle.

準備

  1. Eclipse JEE
  2. 到 Marketplace 找 Gradle 安裝 plugin

套用到既有 project

步驟很簡單：

  1. Create JEE dynamic web project
  2. 建立 build.gradle
  3. 把 dependencies 的 api 找到, 可以到 http://mvnrepository.com/ 找，都有現成的可以 copy
  4. 把 project 轉換成 Gradle project
  5. 在 build.gradle 選擇 reflash all
