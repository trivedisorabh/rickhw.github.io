RoboVM

RoboVM: a toolkit for java developer to create native iOS application.

Requirement

  • Mac OS 10.9
  • Xcode 5.x above
  • JDK 1.7
  • Eclipse 4.2 (Juno)

Installation

  • Find from Marketplace
  • or add update site to install

The current version is RoboVM 0.0.14

Sample

Create a general login page, including account, password, login, reset …

Image in iPhone Simulator

Debug

  • The block point is not work. You cannot step in/out/into in runtime.
  • System out is ok, object info of CocoaTouch is clarify.

Issues

  • Cannot export as ipa file via RoboVM Tools (Right click on RoboVM project), I have no idea what Signing identity is …

Update RoboVM in Eclipse

  1. Eclipse Marketplace
  2. Find with keyword “RoboVM”
  3. Update
