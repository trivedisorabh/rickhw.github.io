Java9 Features

Java9 主要的新功能：

  • modules: 就是一堆 package 的集合. 像是:
    • java.base 就包含了一堆 java.lang.*, java.io.*, java.math … 等 package
    • java.sql: java.sql. javax.sql
    • java.logging
    • java.xml
    • module 跟 package 一樣，也會有相依性的問題。。。這應該會帶來新的問題 orz …
    • 更多可以看這篇： The State of the Module System
  • jshell: cli 工具. 讓 java 也提供項 python/ruby/node 那種對話式的 command line (REPL, Read-Eval-Print-Loop).
  • JMH – a Java harness for building, running, and analyzing nano/micro/milli/macro benchmarks. 分析工具
  • 用 G1 作為 GC 預設的演算法.
  • 支援 HTTP 2.0, 同時更新 HTTP Client API, 實作 websocket.
