Synergy - Soft KVM

以前工作就喜歡用 Synergy 這套來取代一大堆線的 KVM，最近終於有機會再次使用。不過這次的 Client / Server 是不一樣的 OS，試了一下也都沒問題。筆記一下。

設定現在簡單到爆，看圖說故事：

Server - Mac OS (10.9.2)

Client - Windows 7

優點

  • 不用接一堆線 – 主打功能
  • Mac 和 Windows 之間的輸入法切換沒問題。不過在 client (我這邊是 Windows 7) 要切換輸入法之前， server 要先切回 english.
  • config 變得超簡單, 拖拉, 設定 ip 就搞定

缺點：

  • clipboard 有時候會不 work
  • 還是不支援檔案的 DnD …
