AWS SAA-001 Update (Feb, 2018)

AWS Certified Solutions Architect 更版，來自 AWS Blog 的訊息: New AWS Certified Solutions Architect – Associate Exam: Now in General Availability.

正在準備考試的、或者有參考 我的筆記 的人，要注意此資訊。

考試基本資訊

考試要知道的資訊整理如下：

  • exam guide: 考試說明
  • sample exam questions: 範例考題 (PDF)
  • practice exam: 線上試考

這跟過往都差不多，但是可以掌握整個考試方向。

版差差異

AWS SAA 更版，考試編號為 (SAA-01)。考試的方向也更改了，主要以 Well-Architected 五個 Pillars 為考試主軸，項目以及比重如下：

  • Designing resilient architectures - 34%
  • Defining performant architectures - 24%
  • Specifying secure applications and architectures - 26%
  • Designing cost-optimized architectures - 10%
  • Defining operationally excellent architectures - 6%

比重以 resilient 的最高、Security 和 Performant 次之。

Whitepaper 數量變少了，建議閱讀從原本九篇變成以下兩個部分：

  • Architecting on the Cloud - Best Practice - 以 Feb 2016 版為主
    • 同樣標題，還有另一個 Jan 2012 年版的
  • AWS Well-Architected: 這是 2016 re:Revent 加入第五個 Pillar 之後，新增的官方架構頁面，深度提供每個 Pillar 的資訊以及實務案例。

實際上的數量，光是 Well-Architected 頁面是多到不行，所以表面上數量變少了 …

剛好不久前，我在公司內部教育訓練，導讀過這兩篇 Whitepaper.

正在準備的人

整體來說，跟我考的時候，表面上 範圍變小了，但是實際上是更大 + 深度更深。

建議把準備方向：

  • 熟讀 Whitepapers:
    • Whitepaper - Architecting on the Cloud - Best Practice
    • AWS Well-Architected: 五個 Pillar 都要熟讀，以及找到對應的 AWS Services，考量的 Design Priciples, Best Practice … 等
  • 多做 Lab: 這是一定的，沒 Lab 考試絕對不會過
  • Youtube 要多看得系列：
    • Deep Dive
    • This is my architecture

我的準備心得 應該還是有參考價值，但如果只是為了 pass 考試，有些 whitepaper 可以不用讀。

公告上有提到，到今年八月 (Aug, 2018) 都還可以考前一個版本。
